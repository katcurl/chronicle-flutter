# Chronicle v0.9 — Adaptive Desktop Shell

## Цель

Сохранить удобный мобильный интерфейс и добавить самостоятельную компоновку для широких окон без дублирования бизнес-логики.

## Компоновки

- до 760 px: нижняя `NavigationBar`;
- 760–1079 px: компактная `NavigationRail`;
- 1080–1359 px: расширенная `NavigationRail` с подписями;
- от 1360 px: навигация, основная рабочая область и правая контекстная панель.

## Контекстная панель

Показывает:

- активный таймер;
- учтённое время за день;
- число активных задач;
- последние временные сессии;
- подсказки по горячим клавишам;
- local-first статус.

## Горячие клавиши

- `Ctrl/Cmd + 1–5` — переход между разделами;
- `Ctrl/Cmd + T` — открыть запуск таймера.

## Платформы

Сам UI уже адаптивен и тестируется при разных размерах окна. Папки платформ создаются стандартной командой Flutter на соответствующей ОС:

```bash
flutter create --platforms=linux .
```

На Windows в PowerShell:

```powershell
flutter create --platforms=windows .
```

На macOS:

```bash
flutter create --platforms=macos .
```

После генерации платформ общий Dart-код приложения остаётся тем же.
