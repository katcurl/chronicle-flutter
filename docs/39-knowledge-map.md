# Chronicle 0.23 — карта знаний

Версия 0.23.0 добавляет отдельную интерактивную карту существующих связей между заметками.

## Принцип безопасности

Карта является только представлением уже сохранённых заметок и `NoteLink`-записей. Она не изменяет Markdown, не перестраивает ссылки, не переносит файлы и не создаёт новую схему базы данных.

## Возможности

- группировка заметок по проектам;
- панорамирование и масштабирование;
- фильтр по проекту;
- подсветка заметок по названию, тегу или папке;
- отображение числа видимых, внешних и неразрешённых связей;
- открытие заметки непосредственно из карты;
- подсветка рёбер при наведении на узел на настольных платформах.

Размещение узлов вычисляется детерминированно один раз при перестроении экрана. Физическая симуляция и покадровая анимация не используются, поэтому карта не должна влиять на плавность редактора.
