# Chronicle 0.27.0 — визуальная карта исследования

Версия 0.27.0 добавляет свободное визуальное полотно поверх существующего графа знаний. Граф продолжает показывать связи, извлечённые из Markdown, а карта исследования хранит пользовательскую композицию и ручные смысловые связи.

## Возможности

- до 12 независимых исследовательских карт;
- добавление нескольких заметок через поиск и фильтр проекта;
- перетаскиваемые карточки заметок с живыми названиями, тегами и фрагментами текста;
- свободные текстовые карточки для гипотез, вопросов, наблюдений и выводов;
- цветные смысловые области для визуальной группировки;
- ручные направленные связи между любыми карточками;
- изменение размера, дублирование и удаление карточек только с полотна;
- автоматическая раскладка карточек сеткой;
- масштабирование и панорамирование большого полотна;
- открытие исходной заметки двойным щелчком;
- переход между картой знаний и картой исследования.

## Хранение

Исследовательские карты сохраняются в локальных настройках `SharedPreferences` под ключом `chronicle_research_canvases_v1`.

Они не изменяют:

- Markdown заметок;
- связи `[[id:...]]`;
- базу данных Chronicle;
- Vault и его формат;
- вложения;
- LAN-синхронизацию.

Ручная линия на карте является визуальной связью только внутри конкретного полотна. Для устойчивой семантической связи между заметками следует использовать обычную `id:`-ссылку Chronicle.

## Безопасность данных

Удаление карточки заметки с карты не удаляет саму заметку. Удаление карты удаляет только локальную композицию карточек. Проекты, заметки и файлы остаются без изменений.
