# Настраиваемая стартовая страница заметок

Версия: `0.27.3+77`.

## Назначение

Раздел заметок теперь имеет два представления:

- **Обзор** — персональная стартовая страница для продолжения текущей работы;
- **Все заметки** — прежняя библиотека с поиском, фильтрами проекта, папки и закрепления.

Обзор не создаёт отдельную базу и не копирует заметки. Он вычисляется из уже существующих проектов, заметок, задач, папок и шаблонов.

## Доступные секции

1. **Продолжить работу** — активная заметка таймера, записи с незавершёнными задачами и последние изменённые заметки.
2. **Закреплённые** — заметки с флагом `pinned`.
3. **Недавние** — последние изменённые записи по всей базе знаний.
4. **Проекты** — активные проекты с количеством заметок и последней записью.
5. **Папки** — наиболее используемые пути папок с количеством заметок.
6. **Шаблоны** — быстрый запуск создания заметки с заранее выбранным шаблоном.

## Настройка

Кнопка настройки в режиме обзора позволяет:

- менять порядок секций перетаскиванием;
- скрывать отдельные секции;
- выбирать 2, 4, 6 или 8 элементов на секцию;
- включать компактные карточки;
- выбирать, открывать ли раздел заметок с обзора или сразу с библиотеки.

Настройки сохраняются локально в `SharedPreferences` под ключом `chronicle_note_home_preferences_v1`.

## Границы изменения

Функция не меняет:

- Markdown и метаданные заметок;
- формат Vault;
- структуру базы данных;
- синхронизацию;
- рабочие пространства;
- тему;
- профили редактора и панели быстрых действий.
